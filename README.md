# mongo-crud

A CRUD API built with MongoDB and Express.

# Setup

Edit the example.env file and save a copy as ".env".

# Build

`npm run build`

# Run

`npm run production`

# Test

`npm test`



# Sample Get 

`localhost:80/api/documents/`

# Sample Post

URL:
`localhost:80/api/documents`

Header:
`Content-Type = application/json`

Payload:
`{
    "title" : "Title 1",
    "username" : "mat",
    "body" : "This is the body."
}`
